日语教材里常说的“造句情景”，其实大多数是 **动词形态 + 语法功能（语义类别）**。很多教材会按“**表达什么意义/情景**”来分类，比如：意向、样态、推量、原因等。给你整理一个比较完整、学习者常见的分类体系（基本覆盖N3-N1常见语法）。

---

# 一、意志 / 意向类（表示打算、计划、愿望）

表达“想做 / 打算做 / 决定做”。

|表达|例子|含义|
|---|---|---|
|意向形（〜よう / 〜おう）|行こう|我们去吧 / 我打算去|
|〜つもりだ|行くつもりだ|打算去|
|〜予定だ|行く予定だ|计划去|
|〜ことにする|行くことにする|决定去|
|〜たい|行きたい|想去|
|〜たいと思う|行きたいと思う|我觉得想去|

---

# 二、样态 / 推测类

表示“看起来、似乎、可能”。

|表达|例子|含义|
|---|---|---|
|样态そうだ|雨が降りそうだ|看起来要下雨|
|传闻そうだ|雨が降るそうだ|听说会下雨|
|〜らしい|彼は学生らしい|好像是学生|
|〜ようだ|雨のようだ|好像下雨|
|〜みたい|雨みたい|好像下雨|
|〜かもしれない|雨かもしれない|可能下雨|
|〜はずだ|来るはずだ|按理应该来|

---

# 三、可能 / 能力

表示“能不能做”。

|表达|例子|含义|
|---|---|---|
|可能形|行ける|能去|
|〜ことができる|行くことができる|能够去|
|〜得る / 〜得ない|起こり得る|可能发生|

---

# 四、义务 / 必须

表示“必须做”。

|表达|例子|含义|
|---|---|---|
|〜なければならない|行かなければならない|必须去|
|〜なくてはいけない|行かなくてはいけない|必须|
|〜べき|行くべき|应该|
|〜ことだ|勉強することだ|应该做|

---

# 五、许可 / 禁止

表示“可以 / 不可以”。

|表达|例子|含义|
|---|---|---|
|〜てもいい|行ってもいい|可以去|
|〜てはいけない|行ってはいけない|不可以去|
|〜ちゃだめ|行っちゃだめ|不行|

---

# 六、愿望 / 希望

表示希望别人做。

|表达|例子|含义|
|---|---|---|
|〜てほしい|来てほしい|希望你来|
|〜てもらいたい|来てもらいたい|希望你帮忙来|
|〜といい|雨が降らないといい|希望不要下雨|

---

# 七、条件

表示“如果”。

|表达|例子|含义|
|---|---|---|
|〜と|春になると暖かい|一到春天就|
|〜ば|行けば分かる|如果去就知道|
|〜たら|行ったら分かる|如果去了|
|〜なら|行くなら早く|如果去的话|

---

# 八、原因 / 理由

表示“因为”。

|表达|例子|含义|
|---|---|---|
|〜から|雨だから行かない|因为下雨|
|〜ので|雨なので行かない|因为|
|〜ため|雨のため中止|因为|
|〜せいで|雨のせいで|因为（负面）|

---

# 九、目的

表示“为了”。

|表达|例子|含义|
|---|---|---|
|〜ために|勉強するために|为了|
|〜ように|忘れないように|为了不忘|

---

# 十、变化 / 状态

表示状态变化。

|表达|例子|含义|
|---|---|---|
|〜ようになる|日本語が話せるようになった|变得能说|
|〜ようにする|毎日運動するようにする|努力做到|
|〜てくる|暖かくなってきた|逐渐变暖|
|〜ていく|人が減っていく|逐渐减少|

---

# 十一、授受

表达“给 / 帮忙”。

|表达|例子|
|---|---|
|〜てあげる|教えてあげる|
|〜てくれる|教えてくれる|
|〜てもらう|教えてもらう|

---

# 十二、被动 / 使役

语态变化。

|表达|例子|
|---|---|
|被动|食べられる|
|使役|食べさせる|
|使役被动|食べさせられる|

---

# 十三、程度 / 强调

表示程度。

|表达|例子|
|---|---|
|〜ほど|死ぬほど忙しい|
|〜くらい|泣くくらい嬉しい|
|〜ばかり|食べてばかり|

---

# 十四、假设 / 反事实

|表达|例子|
|---|---|
|〜ばよかった|行けばよかった|
|〜のに|行けばいいのに|

---

💡 **核心规律（很重要）**

日语教材里的“情景分类”其实可以压缩成 **5个核心语义系统**：

1️⃣ **意志 / 愿望**  
2️⃣ **推量 / 判断**  
3️⃣ **条件 / 因果**  
4️⃣ **义务 / 许可**  
5️⃣ **变化 / 状态**

几乎所有语法都能归到这五类。
