
## 规律

- 以「す」结尾的 99% 都是他动词
## 清单

| **自动词（搭配 が）** | **他动词（搭配 を）** | **中文核心意思** |
| ------------- | ------------- | ---------- |
| 開く（あく）        | 開ける（あける）      | 开          |
| 閉まる（しまる）      | 閉める（しめる）      | 关          |
| 出る（でる）        | 出す（だす）        | 出 / 拿出     |
| 入る（はいる）       | 入れる（いれる）      | 进 / 放入     |
| 消える（きえる）      | 消す（けす）        | 熄灭 / 消除    |
| 落ちる（おちる）      | 落とす（おとす）      | 掉落 / 使掉落   |
| 壊れる（こわれる）     | 壊す（こわす）       | 坏掉 / 弄坏    |
| 始まる（はじまる）     | 始める（はじめる）     | 开始         |
