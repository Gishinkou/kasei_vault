## 区分自动词和他动词的两种状态

|类型|用法|意思|
|---|---|---|
|自然发生的状态|自动词 + ている|门开着（自然）|
|人为安排的结果|他动词 + てある|门被人打开着|
自动词 + ている
= state

他动词 + てある
= do() → state persists
## 规律

- 以「す」结尾的 99% 都是他动词
